## regra de manutencao

- `README.md` vende a entrada curta.
- `docs/` explica conhecimento estavel.
- `.upstream/` continua sendo a fonte primaria do produto.

Se um documento nao ajuda a clonar, rodar, testar ou entender a demo, ele nao deve permanecer como superficie ativa no repositorio publico.
